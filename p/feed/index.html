Теплофид

Теплофид – это яркие вдохновляющие записи, собранные из множества источников в одну ленту. Выберите любимые категории и наслаждайтесь качественным постами каждый день!

В этом приложении для Android вас ждут:

  • Лучшие записи из ВКонтакте и Tumblr
  • Тематические категории
  • Сохранение любимых записей в Избранное
  • Возможность лайкать и репостить записи из ВКонтакте

Теплофид не требует авторизации для просмотра ленты. Откройте для себя новый источник вдохновения уже сейчас!
